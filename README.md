# Neuromorphic Neuron-Synapse Circuit

## Overview
This project focuses on the design and simulation of a CMOS-based neuron and synapse architecture inspired by biological neural networks. The design was implemented using Cadence Virtuoso and GPDK 90nm technology to emulate neuron spiking behavior and synaptic functionality.
## Objectives
- Design biologically inspired neuron circuits
- Implement CMOS-based synapse structures
- Analyze neural firing characteristics
- Evaluate noise and transient performance
